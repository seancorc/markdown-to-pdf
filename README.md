# Markdown to PDF

Paste Markdown, see a styled live preview, and download a pixel-perfect PDF in one click. Documents autosave in your browser.

## Run it

```bash
npm install
vercel dev
# then visit the printed localhost URL
```

`vercel dev` is needed (not a plain static server) because the PDF is generated by a serverless function. Locally it uses your installed Chrome; on Vercel it uses a bundled serverless Chromium. Set `CHROME_PATH` if your Chrome isn't at the default macOS location.

## How it works

- The front end is a single `index.html`: [marked](https://marked.js.org/) renders Markdown to HTML and [highlight.js](https://highlightjs.org/) highlights code (both via CDN). Documents autosave to `localStorage`.
- Clicking **Download PDF** POSTs the rendered HTML (plus the page's own CSS) to `api/pdf.js`.
- That function renders the HTML with headless Chrome (`puppeteer-core` + `@sparticuz/chromium`) and returns a real PDF. Because it's a true browser engine in print mode, the PDF matches the on-screen preview exactly, with selectable vector text.

## Deploy

Pushing to `main` triggers the linked Vercel project to build and deploy (it installs the dependencies and bundles the serverless function). No environment variables required.
